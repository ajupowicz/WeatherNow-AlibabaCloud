# WeatherNow

A simple, academic-style weather web app.
- Frontend: static HTML/CSS/JS (OSS)
- Backend: Python (Flask) on Alibaba Cloud Function Compute
- Data: OpenWeatherMap

## Run locally
See `backend/README.md` and open `frontend/index.html`. Set `window.BACKEND_BASE_URL` if backend runs on a different port/domain.
